HTML Tags Guide

Tags Functions
h1-h6 Headings. The greater the number next to 'h' the bigger the size of letters.
p Paragraph tag. Commonly used to insert text.
div 'Div' is a container tag. Helps with organizing the structure of html file and later with styling.
nav This element represents a section of a page whose purpose is to provide navigation links, either within the current document or to other documents.
span This element is a generic inline container for phrasing content, which does not inherently represent anything.
img This element helps to insert an image into html file.
a This element helps to insert an external link to file.
embed This element helps with embedding images, videos, or audios to html file.

Styling with HTML

Tag Functions
strong Makes the text appear in bold style.
em Makes the text appear in italic style.
